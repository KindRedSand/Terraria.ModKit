# Terraria.Modkit

Cheat mod for vanilla version of the game which utilize in-game "Journey mode".

## How to use with Steam

- Place mod in to game folder
- In game custom launch params enter full path to mod file:
- - C:\path\to\steamapps\common\Terraria\Terraria.ModKit.exe -steam %command%

## Used code

### [Modders.Toolkit](https://github.com/JavidPack/ModdersToolkit)

- REPL feature
- Area highlighting for copy-paste tool

### [HERO's mod](https://github.com/JavidPack/HEROsMod)

- Map reveal feature
- Map teleport feature

### osu.Framework

- Configuration storage for keybinds
- Logger
